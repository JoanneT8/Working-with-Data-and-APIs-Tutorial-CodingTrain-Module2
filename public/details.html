Project Details of Selfie Data App

Sources

This project is a result of following Coding Train's "Working With Data and APIs in Javascript" tutorial course. The Data Selfie App concept/project idea is taken from Joey K. Lee.

Learning to use the following:

  • node.js
  • Utilization of Bootstrap wesbite template - accessing through CDN link
  • Express to host web server
  • Using Javascript to run a basic server and web app
    • Client side Javascript programming
    • Server side Javascript programming
  • Basic HTML and CSS to create user interface
  • Javascript APIs to pull and utilize different frameworks and information to make this "Selfie Data App" work
    • Geolocation API on client side
    • Fetch API
  • Utilizing git to track and save project progress
    • Git Bash
  • Deploying my server that is being run locally to the cloud so that it can be accessed by other people
  • Using Navigator Web API
  • Getting started with databases - saving the longitude and latitude in the database
      • Step 1: Create an array and store the information there
      Step 2: Look at node File System, save the information to a text file, json file, CSV file

What is static content?

Static content is any content that can be delivered to the user without any processing or modification. The server will deliver the same file to any user who comes to this web page.
